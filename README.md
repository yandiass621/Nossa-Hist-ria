💖 Gerador de Página Dedicatória

Este projeto é uma aplicação web que permite criar uma página personalizada de dedicatória, ideal para compartilhar momentos especiais como aniversários, relacionamentos ou homenagens.

A aplicação possibilita que o usuário personalize diversos elementos da página, como título, mensagem, fotos, música e emojis, enquanto visualiza tudo em tempo real através de um preview interativo.

O objetivo do projeto é proporcionar uma experiência simples e intuitiva para gerar páginas emocionais e personalizadas sem a necessidade de conhecimentos técnicos.

🧩 Funcionalidades

O projeto possui as seguintes funcionalidades:

✏️ Personalização da Página

Definição de título da página

Inserção de mensagem personalizada

Escolha de data especial exibida no preview

🖼️ Upload de Imagens

Upload de até 8 imagens

Miniaturas com opção de remover fotos

Carrossel de imagens no preview

Rolagem automática das fotos

🎵 Música Integrada

Inserção de link do YouTube

Música incorporada automaticamente na página

💖 Emojis Animados

Emojis personalizados que caem pela tela em animação

📱 Preview em Tempo Real

Simulação de visualização dentro de um mockup de celular

Atualização automática enquanto o usuário preenche os campos

⚠️ Alertas Personalizados

Sistema de alert moderno ao invés do alert padrão do navegador

🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais do desenvolvimento web:

HTML5 → Estrutura da aplicação

CSS3 → Estilização moderna, layout responsivo e animações

JavaScript (Vanilla JS) → Interatividade, manipulação do DOM e lógica da aplicação

Outros recursos utilizados:

Google Fonts (Inter)

YouTube Embed API

Animações CSS

Manipulação dinâmica de imagens

🎯 Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de:

Praticar manipulação de DOM com JavaScript

Trabalhar com eventos e interatividade

Criar interfaces modernas com CSS

Desenvolver um preview dinâmico em tempo real

Além disso, o projeto também demonstra conceitos importantes como:

gerenciamento de arquivos enviados pelo usuário

integração de conteúdo externo (YouTube)

animações e efeitos visuais

📱 Responsividade
